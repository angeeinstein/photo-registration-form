üîß OAuth 2.0 Setup Instructions

  1. Go to Google Cloud Console:
    • Visit Google Cloud Console - Credentials
    • Select your project (or create one)
  2. Enable Google Drive API:
    • Go to Drive API Library
    • Click "ENABLE"
  3. Create OAuth 2.0 Client ID:
    • Back in Credentials, click "+ CREATE CREDENTIALS"
    • Select "OAuth 2.0 Client ID"
    • If prompted, configure OAuth consent screen first
    • Application type: "Web application"
    • Name: Photo Registration OAuth
  4. Configure Authorized Origins:
    • Under "Authorized JavaScript origins", click "+ ADD URI"
    • Add: http://localhost:5000
    • Use your actual domain (http:// or https://)
  5. Leave Redirect URIs Empty:
    • For popup flow (recommended), leave "Authorized redirect URIs" empty
    • Or add your domain URL if needed
  6. Create and Copy Credentials:
    • Click "CREATE"
    • Copy the Client ID
    • Copy the Client Secret
  7. Update .env File:
    • Open your .env file
    • Add these lines:
    GOOGLE_OAUTH_CLIENT_ID=your-client-id.apps.googleusercontent.com
GOOGLE_OAUTH_CLIENT_SECRET=GOCSPX-your-client-secret
GOOGLE_OAUTH_REDIRECT_URI=postmessage
  8. Restart Application:
    • Stop the application (Ctrl+C or systemctl stop)
    • Start it again
    • Refresh this page
  9. Connect:
    • Click "Connect Google Drive" button above
    • Sign in with your Google account
    • Grant Drive permissions
    • ‚úÖ Done!
